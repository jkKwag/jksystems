# 🚐 CampRoad - 캠핑카 예약 관리 앱

캠핑카 소개, 예약, Q&A, FAQ, 관리자 기능을 갖춘 React 앱입니다.

## 기능

- 🏕 **캠핑카 소개** - 카드 리스트 + SVG 지도 보기
- 📋 **예약 시스템** - 날짜 선택, 인원, 요금 자동 계산
- 💬 **Q&A** - 문의 작성 및 답변 확인
- ❓ **FAQ** - 자주 묻는 질문
- ⚙️ **관리자** - 관리자 로그인 페이지

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm start
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 접속

### 빌드

```bash
npm run build
```

## 기술 스택

- **React 18**
- **JavaScript (JSX)**
- CSS-in-JS (인라인 스타일)

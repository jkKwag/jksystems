기능별 설명

🔍
QR 기반 메뉴 접근
카메라로 QR 스캔 시 URL 파싱하여 사업장 식별 및 테이블 번호 추출. 앱 설치 불필요, 브라우저에서 즉시 실행.
🤖
AI 메뉴 추천 (Claude API)
Anthropic Claude API 연동. 자연어 대화·음성인식으로 메뉴 추천 및 장바구니 자동 추가. 예약 신청도 AI 대화로 처리.
🪑
좌석 선택 예약
좌석 그리드 UI에서 시각적으로 좌석 선택. 날짜(캘린더)·시간·인원 설정 후 예약 신청. 사업자 승인/거절 처리.
💬
예약 실시간 채팅
예약번호로 채팅방 입장. Supabase Realtime WebSocket 기반. 동일 예약 인원 간 그룹 채팅 지원.
💳
간편 결제 (TossPayments)
TossPayments SDK v2 연동. 매장주문/포장주문 선택 후 결제. 성공·실패 페이지 별도 처리, 장바구니 자동 초기화.
📋
개인정보 수집 동의
주문·예약 시 개인정보 수집·이용 동의 절차 내장. 동의 내역 Supabase에 기록 (tb_usr_prv_cns).
